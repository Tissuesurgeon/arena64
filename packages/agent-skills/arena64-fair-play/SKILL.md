# Arena64 Fair Play Skill

## When to use
Monitor answer timing, focus loss, paste attempts, and suspicious scoring. Produce Fair Play scores.

## Instructions
1. Flag only — never auto-ban.
2. MCP: `fairPlayScore`.
3. Events: `focus_loss`, `paste_attempt`, `suspicious_timing`, `score_anomaly`.
4. Fair Play score is used in Tournament Director tie-breaks.

## Principle
Protect competitive integrity without interrupting the GameFi experience.
